# ARP Parser
Программа для парсинга и вывода полей ARP-пакетов (Address Resolution Protocol).

## Описание
Arp_parser принимает на вход бинарные данные, представляющие ARP-пакет, разбирает его структуру и выводит значения полей в читаемом виде.

## Поддерживаемые поля ARP-пакета
| Поле       | Описание      | Размер (байт) |
| ---------- |:-------------:|:-------------:|
| htype      | Тип аппаратного адреса      | 2|
| ptype      | Тип протокола               | 2|
| hsize      | Размер аппаратного адреса   | 1|
| psize      | Размер протокольного адреса | 1|
| op         | Код операции                | 2|
| sha        | MAC-адрес отправителя       | 6|
| spa        | IP-адрес отправителя        | 4|
| tha        | MAC-адрес получателя        | 6|
| tpa        | IP-адрес получателя         | 4|


# Erlang
## Структура модуля 
parse/1 - основная функция парсинга. Принимает бинарные данные, возвращает ok и выводит информацию о пакете.

print_arp/1 - выводит разобранный ARP-пакет.

format_mac/1 - преобразует бинарный MAC-адрес в строку формата XX:XX:XX:XX:XX:XX.

format_ip/1 - преобразует бинарный IP-адрес в строку формата XXX.XXX.XXX.XXX.

test/0 - тестовая функция, использующая тестовый ARP-пакет.

# C
## Структура
arp.c - содержит основные функции для парсинга и вывода.

arp.h - заголовочный файл со структурой.

arp_types.h - содержит значения типов адресов, протоколов и операций. Взято с www.iana.org.

main.c - простой запуск одного тестового пакета.

test.c - содержит в себе различные тесты.

## Цели make
main - компиляция с main.c
test - компиляция с test.c
run - запускает test и main
и остальные базовые.
